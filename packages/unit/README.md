# @hidoo/unit

> Sass based css framework.

## Installation

```sh
$ npm install @hidoo/unit
```

## Usage

```scss
@import "node_modules/@hidoo/unit/src/index.scss";
```

### Usage with [node-sass-magic-importer](https://github.com/maoberlehner/node-sass-magic-importer)

```scss
@import "~@hidoo/unit";
```

## Test

```sh
$ yarn test
```
