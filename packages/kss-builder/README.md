# @hidoo/kss-builder

> Custom builder for kss-node.

## Installation

```sh
$ npm install --save-dev kss @hidoo/kss-builder
```

## Usage

```bash
$ kss path/to/css path/to/dest --builder @hidoo/kss-builder
```
