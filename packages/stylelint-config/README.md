# @hidoo/stylelint-config

> Sharable stylelint config for my projects.

## Installation

```sh
$ npm install --dev stylelint @hidoo/stylelint-config
```

## Usage

```js
{
  extends: [
    '@hidoo/stylelint-config'
  ]
}
```

### Usage in SCSS syntax

```js
{
  extends: [
    '@hidoo/stylelint-config/scss'
  ]
}
```

## Test

```sh
$ yarn test
```
